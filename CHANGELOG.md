# Changelog

## v0.0.7 - 2026-02-18

- Added manual and automatic annotation keyframe modes, with UI updates and related shortcut/settings support.
- Improved crop/export performance by using stream-copy when possible to avoid unnecessary re-encoding.
- Introduced a centralized `AppPalette` theme system and applied cohesive styling across player, timeline, loop, crop, and annotation UI.

## v0.0.6 - 2026-02-17

- Added reliable fullscreen toggle support with `F11` and `Esc`.
- Added annotation-aware export flow and improved viewport/video sizing consistency.

## v0.0.5 - 2026-02-16

- Added fullscreen mode support for faster preview workflow.
- Added annotation keyframe timeline and related export/migration improvements.
- Added export range controls with more robust FFmpeg export behavior.

## v0.0.4 - 2026-01-28

- Added text annotation tool.
- Added grouped shortcut toggle controls.

## v0.0.3 - 2026-01-26

- Added selection and shape tools (rectangle, circle, line, arrow) to the annotation workflow.
- Refactored formatting and simplified widget constructors.

## v0.0.2 - 2026-01-26

- Added Windows file association support and setup guidance.
- Expanded loop and crop workflow with keyboard shortcuts.

## v0.0.1 - 2026-01-19

- Initial release.
